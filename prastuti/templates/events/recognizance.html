Description

With an abundance of data and data-driven companies like Google and Facebook continuing to analyze and research intelligent machine algorithms, The domain of Machine learning has boomed like never before. Recognizance is a machine learning hackathon that thrives to give a real-world experience with machine learning and its implementations. This provides an opportunity to learn about the fundamental algorithms that are "state-of-the-art" solutions. The algorithms would mainly comprise of Supervised regression and classification. We mainly focus on simple Neural Networks, Computer Vision and NLP. To top it off it would be competition to squeeze out the least error and highest accuracy.

And may the Gradients be with you.
